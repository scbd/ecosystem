# Design Ecosystem

> Design system  built with nuxtjs & vuepress

## Build Setup

``` bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn  run dev

# build for production and launch server
$ yarn  run build
$ yarn  start
```
